Project Overview

This page records the technical projects I have built while developing my programming, web development and software design skills. The projects include Python command-line applications, HTML and CSS layouts, JavaScript mini apps and university programming artefacts.

These projects show my progression from basic syntax and layout practice to more structured programs using functions, data structures, input validation, local storage, responsive design and debugging.

Cozy Calculator Web App

A JavaScript calculator with a soft, cozy interface. The project uses HTML, CSS and JavaScript to handle arithmetic operations, keyboard input, clear and backspace controls, calculation history and localStorage.

  • Skills: JavaScript functions, DOM manipulation, events and localStorage
  • Features: calculation history, clear history button, keyboard support and animated cloud styling
  • Learning: debugging broken functions, fixing HTML structure and improving UI polish

Python Financial Tracker

A university Python project that allows users to add, view, delete, filter and summarise income and expense transactions. The program uses a list of dictionaries to store transaction records clearly.

  • Grade: 78%
  • Skills: Python, functions, loops, dictionaries, lists and input validation
  • Features: category filtering, date validation, balance summaries and highest/lowest expense calculations
  • View Python Repository

The Bunny and the Golden Key

A Python text-based adventure game where the player navigates rooms, collects items, avoids traps and searches for the Golden Key before the burrow collapses. Naturally, the bunny had a more organised quest structure than half the internet.

  • Skills: Python dictionaries, lists, functions, loops and conditionals
  • Features: room map, inventory, hidden traps, move counter, hints and replay loop
  • Learning: game state tracking, debugging edge cases and testing player commands
